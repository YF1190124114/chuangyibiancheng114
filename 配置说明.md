# 四季树生成器 - 配置文件说明

## 📁 配置文件位置
`data/config.json` - 所有季节的配置都在这个文件中

## 🎮 使用方法
1. 运行程序后，按 **1-4** 键切换季节
2. 修改 `data/config.json` 文件中的数值
3. 重新运行程序或切换季节即可看到效果

## 🌸 春天配置 (spring)
```json
"spring": {
  "branchColor": { "r": 101, "g": 67, "b": 33 },     // 树枝颜色 (RGB)
  "leafColorMin": { "r": 144, "g": 238, "b": 144 },  // 叶子颜色最小值
  "leafColorMax": { "r": 200, "g": 255, "b": 200 },  // 叶子颜色最大值
  "background": { "r": 240, "g": 248, "b": 255 },    // 背景颜色
  "leafCount": 3500,        // 最大叶子数量
  "minLeafCount": 2500,     // 最少叶子数量
  "backfillPerFrame": 150,  // 每帧补充叶子数量
  "alongLeafRadius": 35,    // 叶子分布半径
  "lsystem": {
    "iterations": 3,        // 迭代次数
    "angleDeg": 25,         // 分支角度
    "baseLen": 85,          // 基础长度
    "lenFactor": 0.7,       // 长度缩放
    "trunkBoost": 1.6,      // 主干增强
    "trunkThickBoost": 1.2  // 主干粗细增强
  },
  "thickness": { "start": 5, "decay": 0.98, "min": 1 }, // 线宽设置
  "ground": {               // 地形设置
    "baseOffset": 6,
    "noiseAmp": 12,
    "noiseScale": 0.01,
    "step": 4,
    "seed": 1337
  }
}
```

## 🌿 夏天配置 (summer)
- 深绿色叶子，暖黄背景
- 标准叶子分布和树形

## 🍂 秋天配置 (autumn)
- 金黄橙红叶子，暖橙背景
- 最茂盛的叶子，4次迭代

## ❄️ 冬天配置 (winter)
- 雪白叶子，冷蓝背景
- 最简洁的树形，2次迭代

## 🎨 参数说明

### 颜色参数
- `branchColor`: 树枝颜色 (RGB 0-255)
- `leafColorMin/Max`: 叶子颜色范围，程序会在这个范围内随机选择
- `background`: 背景颜色 (RGB 0-255)

### 叶子参数
- `leafCount`: 最大叶子数量
- `minLeafCount`: 最少叶子数量
- `backfillPerFrame`: 每帧补充的叶子数量
- `alongLeafRadius`: 叶子分布半径，越大越松散

### L-system 参数
- `iterations`: 迭代次数，影响树的复杂程度
- `angleDeg`: 分支角度（度）
- `baseLen`: 基础长度
- `lenFactor`: 每次迭代的长度缩放比例
- `trunkBoost`: 主干长度增强倍数
- `trunkThickBoost`: 主干粗细增强倍数

### 线宽参数
- `start`: 起始线宽
- `decay`: 每步衰减系数
- `min`: 最小线宽

### 地形参数
- `baseOffset`: 离底部的基础高度
- `noiseAmp`: 噪声幅度
- `noiseScale`: 噪声频率
- `step`: 台阶步长
- `seed`: 随机种子

## 💡 修改建议
1. **调整颜色**: 修改 RGB 值 (0-255)
2. **调整叶子数量**: 修改 `leafCount` 和 `minLeafCount`
3. **调整树形**: 修改 `lsystem` 中的参数
4. **调整叶子分布**: 修改 `alongLeafRadius`
5. **调整线宽**: 修改 `thickness` 中的参数

## ⚠️ 注意事项
- 修改后需要重新运行程序或切换季节才能看到效果
- 保持 JSON 格式正确，注意逗号和括号
- 数值范围：颜色 0-255，角度 0-360，其他参数根据需要调整
